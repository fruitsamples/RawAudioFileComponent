### RawAudioFileComponent ###

===========================================================================
DESCRIPTION:

The RawAudioFileComponent project provides a template for creating a custom audio file component. The sample provides an example of how to implement the necessary methods to be a valid audio file system extension.

===========================================================================
BUILD REQUIREMENTS:

Mac OS X v10.6 or later

===========================================================================
RUNTIME REQUIREMENTS:

Mac OS X v10.6 or later

===========================================================================
PACKAGING LIST:

RawAudioFileComponent.cpp
- Main implementation of the component
